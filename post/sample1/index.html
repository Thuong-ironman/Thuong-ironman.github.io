Localisation and Navigation

  1. State Estimation What is a state estimator? There are different definitions, according to the domian of application considered. In a very broad sense, it can be defined as an algorithm that, given a sequence of measurements, reconstructs the internal state of a system. The measurements are collected from sensors. Examples of sensors for UAVs:
  • GPS;
  • Magnetometer;
  • Inertial Measurement Units;
  • Gyroscopes;
  • (Stereo) cameras;
  • RGB-D cameras;
  • Radio-Frequency Systems: Ultra-Wide Band (UWB), Radio Frequency IDentification (RFID);
  • LiDAR
  • And neuromophic sensors (Event cameras) - the one that I am doing research in recent year

Static vs dynamic The state estimates are different depending on the nature of the state and on the number and types of the available sensors. Static estimators are used whenever the state can be determined with one single collections of measurements. In other words, when the sensors give information on all the components of the state. Dynamic estimators are used when the state can be reconstruced using the sensors and some other additional information constraints on the state. Usually, dynamic estimators use sensor readings collected at different time instants and combine them with the state dynamic.
